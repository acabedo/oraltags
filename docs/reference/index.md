# Índice del paquete

## Funciones exportadas

El paquete expone dos funciones; el resto de la funcionalidad vive en la
interfaz Shiny que se lanza con run_app().

- [`run_app()`](https://acabedo.github.io/oraltags/reference/run_app.md)
  : Lanzar la aplicación oraltags
- [`oraltags_data_dir()`](https://acabedo.github.io/oraltags/reference/oraltags_data_dir.md)
  : Carpeta de datos del usuario para oraltags
