# Código de conducta

Este proyecto adopta el **Contributor Covenant** (v2.1) como código de conducta.

Nos comprometemos a ofrecer una experiencia libre de acoso para todas las personas,
con independencia de su identidad o experiencia. Se espera un trato respetuoso y
constructivo en *issues*, *pull requests* y demás espacios del proyecto.

Comportamientos inaceptables incluyen lenguaje o imágenes ofensivas, ataques personales,
acoso público o privado, y la divulgación de información privada de terceros sin permiso.

**Cómo reportar:** si presencias o sufres una conducta inaceptable, contacta con el
responsable del proyecto a través del [issue tracker](https://github.com/acabedo/oraltags/issues)
o por correo a quien mantiene el repositorio. Las incidencias se revisarán de forma
confidencial y se actuará según se considere apropiado.

Texto completo del Contributor Covenant: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
